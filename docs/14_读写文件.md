## Scan 扫描用户输入
- fmt.Scanln  扫描一行数据，到换行符号结束
```go
    var name string
	fmt.Scanln(&name)
```
- fmt.Scanf  按照某个格式扫描
```go
var (
    name   string
    age    byte
)
fmt.Scanf("%s %d", &name, &age)
```

## bufio 包
bufio包实现了有缓冲的I/O。它包装一个io.Reader或io.Writer接口对象，创建另一个也实现了该接口，且同时还提供了缓冲和一
些文本I/O的帮助函数的对象。
